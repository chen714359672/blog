# 自适应方案

# 主流： rem和vh

rem指相对于根元素的字体大小的单位。em是指相对于父元素的字体大小的单位

body,html{
    padding: 0;
    margin: 0;
    max-width: 768px;
    margin: 0 auto;
}
html{
    font-size: calc(100vw / 7.5);
}
@media screen and (min-width: 768px) {
    html{
        font-size: 108px;
    }    
}
#app {
    height: 100%;
    font-size: initial;
}


转换方面的话可以用**postcss-px-to-viewport**自动转换为vw

# 软键盘将页面顶起来、收起未回落问题

关于键盘弹出的话，在安卓上我一般都会进行监听高度变化来做一个相对应的处理

let originHeight = document.documentElement.clientHeight || document.body.clientHeight;
//ios不会触发resize事件
window.onresize = function () {
    let resizeHeight = document.documentElement.clientHeight || document.body.clientHeight;
    if (resizeHeight < originHeight) {
        //键盘弹起
            setTimeout(() => {
            document.querySelector('弹窗遮罩').style.height = document.body.offsetHeight + 'px';
            document.querySelector('弹窗遮罩').style.overflow = 'scroll';
                },0);
            } else {
                //键盘收起
            document.querySelector('弹窗遮罩').style.height = 517 + 'px'; //517为弹窗原先高度
        }
}






## 弹窗遇到滚动穿透

移动端弹出fixed弹窗的话，若底部背景页面存在滚动条，则滑动弹窗会导底部的背景页面跟着滚动，称为“滚动穿透”

若弹窗下层（背景页面上层）还有fixed定位的一遮罩，此时滑动遮罩背景页面也会跟着一起滚动



### 解决方案

```js
watch: {
        'showModal'(val) {
        let ele = document.querySelector('内容超出自身高度的div');
            if(val) {
                 ele.style.overflow = 'hidden';
            } else {
                ele.style.overflow = 'auto';
            }
        }
    }

# 移动端的1px问题

将伪元素的长和宽先放大2倍，然后再设置一个边框，以左上角为中心，缩放到原来的0.5倍

//mixins.less
//上边框
.setTopLine(@clolor) {
    content: " ";
    position: absolute;
    left: 0;
    top: 0;
    right: 0;
    height: 1px;
    background: @clolor;
    transform-origin: 0 0;
    /* 1.5倍屏 */
    @media (-webkit-min-device-pixel-ratio:1.5){
        transform: scaleY(0.6666);
    }
    /* 2倍屏 */
    @media (-webkit-min-device-pixel-ratio:2){
        transform: scaleY(0.5);
    }
    /* 3倍屏 */
    @media (-webkit-min-device-pixel-ratio:3){
        transform: scaleY(0.3333);
    }
}


//四条边框
.setAllLine(@clolor,@radius: 0) {
    content: "";
    position: absolute;
    top: 0;
    left: 0;
    transform-origin: 0 0;
    border: 1px solid @clolor;
    @media (-webkit-min-device-pixel-ratio: 1.5){
        width: 150%;
        height: 150%;
        transform: scale(0.66666);
        border-radius: @radius * 1.5;
    }
    @media (-webkit-min-device-pixel-ratio: 2){
        width: 200%;
        height: 200%;
        transform: scale(0.5);
        border-radius: @radius * 2;
    }
    @media (-webkit-min-device-pixel-ratio: 3){
        width: 300%;
        height: 300%;
        transform: scale(0.33333);
        border-radius: @radius * 3;
    }
}

# 常用的meta属性


<meta name="screen-orientation" content="portrait"> //Android 禁止屏幕旋转
<meta name="full-screen" content="yes">             //全屏显示
<meta name="browsermode" content="application">     //UC应用模式，使用了application这种应用模式后，页面讲默认全屏，禁止长按菜单，禁止收拾，标准排版，以及强制图片显示。
<meta name="x5-orientation" content="portrait">     //QQ强制竖屏
<meta name="x5-fullscreen" content="true">          //QQ强制全屏
<meta name="x5-page-mode" content="app">            //QQ应用模式
